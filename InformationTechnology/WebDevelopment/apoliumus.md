# Media Query

- `@media` rule to include a css property if a certain condition is true.

```
@media only screen and (max-width: 600px) {
  body {
    background-color: lightblue;
  }
}
```

- for conditions:
    - max-width
    - min-width
    - oreintation

- inside `@media` normal css will be applied.
