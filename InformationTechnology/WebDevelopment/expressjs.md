## Rest API Convention

| CRUD Actions     | HTTP Method | Endpoints         |
| ---------------- | ----------- | ----------------- |
| Get all contacts | GET         | /api/contacts     |
| Get contact      | GET         | /api/contacts/:id |
| Create contact   | POST        | /api/contacts     |
| Update contact   | PUT         | /api/contacts     |
| Delete contact   | DELETE      | /api/contacts/:id |
